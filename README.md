# microsoft.onedrive Connector for WSO2 MI

Microsoft OneDrive is a cloud-based file storage service provided by Microsoft, allowing users and organizations to store, share, and manage files securely online.

### Compatibility

| Connector Version | Supported WSO2 MI Version |
|-------------------|---------------------------|
| 0.1.0             | MI 4.5.0, MI 4.4.0        |


## Documentation

Please refer to the [microsoft.onedrive Connector Overview](https://mi.docs.wso2.com/en/latest/reference/connectors/) documentation.

### Building From the Source

Follow the steps given below to build the connector from the source code:

1. Get a clone or download the source from [Github](https://github.com/wso2-extensions/mi-connector-microsoft.onedrive/).
2. Run the following Maven command from the `mi-connector-microsoft.onedrive` directory: `mvn clean install`.
3. The ZIP with the microsoft.onedrive connector is created in the `mi-connector-microsoft.onedrive/target` directory.
